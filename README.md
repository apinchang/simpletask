# SimpleTask - 简单任务管理

一个简洁高效的Chrome浏览器扩展，用于管理日常任务。

## 功能特点

- 📋 **三象限任务管理**：按照重要性和紧急程度将任务分为三类
  - 🔥 重要且紧急
  - ⏳ 重要不紧急
  - 📦 不重要不紧急

- 🎯 **拖拽排序**：使用SortableJS实现流畅的拖拽功能
  - 支持同一分类内任务排序
  - 支持跨分类拖动任务

- 📝 **任务编辑**：双击任务内容即可编辑

- 🗑️ **历史记录**：删除的任务会保存到历史记录，支持恢复或彻底删除

- 📤 **导入导出**：支持JSON格式的数据导入导出

## 安装方法

1. 下载或克隆本项目到本地
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用方法

1. 打开Chrome浏览器新标签页，会自动显示任务管理界面
2. 在输入框中输入任务内容，点击"+"或按回车键添加任务
3. 拖动任务前的☰图标可以调整任务顺序或移动到其他分类
4. 双击任务内容可以编辑任务
5. 点击任务右侧的×可以删除任务

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Chrome Extension API
- SortableJS
